server setup prompt

Read this prompt carefully, then copy and paste it into the agent tool that has shell access to your server. It will walk through cloning the repo, generating or using a config, and running setup safely. Everything is readable to a human.

Setup script testing is focused on ubuntu 24.04, but oqto can run on any linux distro in principle. Ensure your ssh key is installed on the server before running hardening steps (password auth may be disabled). This site does not include hidden html comments or obfuscated prompts. Please always read any text you paste into an agent tool.
